Printed Circuit Board Soldering Defect Identification using Basler Camera and GUI
• Using Windows OS
• Make sure you have the Basler app installed. https://baslerweb.com/en/downloads/software-downloads/ or https://www.baslerweb.com/en/downloads/
• pip install pypylon
• pip install tk
• pip install opencv-python
• pip install Pillow
• run pcb_detection.py https://github.com/charlierolando/Printed-Circuit-Board-Soldering-Defect-Identification-using-Basler-camera/blob/main/source/pcb_detection.py

Collect Data using Basler Camera
• pip install pypylon
• pip install opencv-python
• run collect_data.py https://github.com/charlierolando/Printed-Circuit-Board-Soldering-Defect-Identification-using-Basler-camera/blob/main/source/collect_data.py
